# LkfCoder

mp3->lkf
Приложение автоматически кодирует файлы mp3 в lkf и загружает их на флэшку. lkf - это формат музыкального файла, который используют устройства имеющие название Тифлофлешплеер. В этой версии появилась поддержка многопоточного режима.
